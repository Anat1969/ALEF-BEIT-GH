## 📋 **רשימה מלאה — 22 Skillים**

### **1️⃣ א׳ (אלף) — פתיחת היום**

- **Action:** נשימה × 3 + 3 כוונות למה שרוצים היום

## 🔄 **Workflow שבועי**

### **שבוע 1: א׳–ו׳ (פתיחה → קשר)**
```
יום א׳: א׳, ב׳, ג׳, ד׳, ה׳, ו׳
יום ב׳: חזור על אותו דפוס
... (כל יום אותו דפוס עד שהופכים להרגל)
```

### **שבוע 2: ז׳–יב׳ (עבודה → תשלום)**
```
הוסף י׳, יא׳, יב׳ בהדרגה
בדוק איך השילוב עובד
```

### **שבוע 3+: יג׳–כב׳ (מעמקה → סוף)**
```
הוסף בהדרגה, אל תעמוד ברבות הקפיצות
יום יום = מעגל חדש
```

## 💡 **Design Notes**

**Why this order (א׳–ת׳)?**
- Linguistic flow (Hebrew alphabetic order)
- Energy arc (morning opening → evening closure)
- Archetypal progression (birth → death → rebirth)
- Practical sequencing (physiology of the day)

**Why 22 skills?**
- Not 10 (too few)
- Not 100 (too many)
- 22 = sacred number in Hebrew, full alphabet
- Covers all life domains (work, rest, relationships, creation)
